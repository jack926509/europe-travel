# 🇪🇸🇵🇹🇫🇷 西班牙・葡萄牙・南法 自由行全攻略 2025-2026

這是一份專為手機端優化的歐洲自由行網頁（Web App），目標是作為旅行當下的隨身資料庫，以最舒適的閱讀體驗，提供從規劃到落地所需的完整資訊。

---

## 📱 網站提供資訊總覽

本網站將龐大的旅遊資訊系統化分類為 **5 大核心區塊**，透過直覺的底部導航列，方便旅客在旅途中單手快速查詢：

### 1. 📋 行前準備
*   **簽證與入境**：最新 ETIAS 電子旅行許可及 EES 入境系統上線提醒。
*   **航班選擇**：直飛與轉機（杜拜、伊斯坦堡等）優缺點比較。
*   **實用旅行準備**：申根保險建議、網卡/eSIM 上網方案比較、當地消費習慣與換匯建議。

### 2. 🚆 交通指南
*   **跨國移動**：飛機廉航與跨國巴士/火車的優劣分析。
*   **西班牙交通**：Renfe 國鐵、ALSA 巴士，以及馬德里/巴塞隆納市區地鐵票卡。
*   **葡萄牙交通**：CP 國鐵與里斯本 Navgante 卡、波多 Andante 卡使用方式。
*   **南法交通**：蔚藍海岸與普羅旺斯的自駕建議及 SNCF 區域火車攻略。
*   **實用工具**：必備的交通與導航 App 推薦清單。

### 3. 📖 旅遊攻略
*   **西班牙🇪🇸**：巴塞隆納、馬德里、南部安達魯西亞的必訪景點、Tapas 必吃美食與特色節慶。
*   **葡萄牙🇵🇹**：里斯本、波多的精華景點，以及正宗蛋塔、海鮮燉飯等美食推薦。
*   **法國南法🇫🇷**：蔚藍海岸、普羅旺斯的特色小鎮與必吃南法料理。
*   **🏰 巴黎迪士尼**：特別收錄 2026 新開幕「冰雪奇緣園區」介紹、雙園必玩設施、票價與餐廳攻略。

### 4. 🛍 購物與退稅
*   **精選伴手禮**：三國特色伴手禮推薦（如：西班牙無花果巧克力、葡萄牙花磚、南法馬賽皂等）。
*   **精品購物**：歐洲名牌攻略與各國精品 CP 值分析（Loewe, LV, Chanel 等）。
*   **退稅攻略**：詳列各國退稅門檻（如：西班牙無門檻）、退稅流程，以及最新台灣入境免稅額度（NT$35,000）。

### 5. 💡 實用叮嚀
*   **治安注意**：熱門觀光區扒手手法警示與防竊建議。
*   **行程與天氣**：重點景點（如聖家堂、阿爾罕布拉宮）提前預訂建議與穿搭指南。
*   **在地生存**：小費文化、歐洲罷工應對方式、實用求生語，以及緊急聯絡電話（警察 / 急救 / 台灣駐外辦事處）。

## 📝 修改歷程 (Changelog)

### v2.1 ｜ 2026-03-11 (Mobile UX/UI 全面進化)
*   **介面架構重構**：將一頁式超長版面轉換為「5大分頁系統」，支援無縫切換，大幅減少滾動疲勞。
*   **新增底部導航列 (Tab Bar)**：新增如原生 App 般的底部導航列，具備毛玻璃與高亮選取效果，支援手機大拇指快速切換。
*   **手勢操作支援 (Swipe)**：增加左右滑動（Swipe）手勢以切換主分頁，提升手機使用者互動體驗。
*   **版面微調與壓縮**：
    *   將過長的子導航按鈕精簡化（例如：「跨國移動」縮短為「跨國」、「景點行程」縮短為「景點」）。
    *   縮小按鈕 Padding 與間距，防止在小螢幕手機上破版變形。
*   **文字語氣優化**：將整體的「貼士」全數調整為「叮嚀/叮囑」，更符合台灣讀者語境。
*   **色彩與字體加強**：更換整體色系為更具質感的奢華暖色系（陶土紅、琥珀金），統一字體樣式並增大手機版預設行距，提升強光下的閱讀性。
*   **頁尾 (Footer) 優化**：新增更新版本號、免責聲明、以及重要的 ETIAS/EES 首發預警資訊連結。
